
24. Petit réseau local simple
 Le Switch aura pour rôle d'interconnecter les deux PC se trouvant dans le réseau
afin de les relier au Routeur. Car le Switch nous permet d' interconnecter les
hôtes qui se trouvent dans le même réseau.
 Dans ce réseau, le Routeur va permettre à ce que ce réseau puisse avoir la
possibilité de communiquer avec d'autres réseaux.
25. Réseau avec 2 sous-réseaux
 Le Routeur est nécessaire pour permettre de relier les deux sous-réseaux afin
d'établir une communication entre les PC qui se trouvent dans les différents
réseaux. Par ailleurs il va encore jouer le rôle de passerelle par défaut.
